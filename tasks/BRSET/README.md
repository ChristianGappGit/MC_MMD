# BRSET
2D opthalmological color fundus images / tabular data (patient information)
classification of following 

##13 diseases:
diabetic retinopathy, 
macular edema, 
scar, 
nevus, 
amd, 
vascular occlusion, 
hypertensive retinopathy, 
drusens, 
hemorrhage, 
retinal detachment, 
myoptic fundus, 
increased cup disc, 
other. 

Another is for no finding.
